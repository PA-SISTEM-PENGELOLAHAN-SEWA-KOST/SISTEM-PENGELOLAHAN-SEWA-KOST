# Menu Admin
![Gambar WhatsApp 2025-10-26 pukul 18 19 51_8a6df95f](https://github.com/user-attachments/assets/e9bf3e48-9da5-496b-ad8b-16648aa061a4)
# Menu ini berisi daftar pilihan untuk mengelola data kamar dan riwayat sewa. Administrator dapat Melihat Daftar Kamar, Menambah Kamar baru, Mengubah (Update) detail kamar yang sudah ada, atau Menghapus Kamar. Selain itu, tersedia fungsi untuk melihat Semua Riwayat Sewa dan Mengubah Status Sewa kamar. Setelah selesai, admin dapat memilih Logout untuk keluar dari sesi atau memilih Keluar dari Program untuk menutup aplikasi
